Part One

First and foremost, I learned about the differenc between HTML and CSS. HTML plays the structural role as it is used to put certain content into a certain tag. On the other end, CSS helps style the page to make the page look nice. A good analogy to use is that HTML is the physical body type of a person while the CSS could be thought of as either the clothes and accessories or their personality. JavaScript helps add the functionality of the page. Going back to the analogy, I would say that JavaScript are the actions that a person would do, like a verb.

Here is an exmaple of some basic CSS code to change the background color of a page to light blue.

body { background-color: lightblue; }
