Flexbox

Over the past week I got to learn more about Flexbox and how to use it. Flexbox is a great tool to use to help layout the different divs that I use. It is extremely helpful to line up divs and spans either as a row, column, or somewhere in between. The labs this week helped me pratice with it by making some basic static sites. This tool is extremely useful because it helps with not having to rely on hardcoding the values of the certain element.

.container {
 display: flex;
 flex-direction: row;
 justify-content: flex-start;
 align-items: flex-start;
 height: 100vh;
 background-color: lightgray;
}
